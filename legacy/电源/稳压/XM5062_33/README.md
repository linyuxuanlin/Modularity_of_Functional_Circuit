# 基本参数

* Input: 2.5V to 5.5V
* Output: 3.3V
* Current: 600mA
* Efficiency: Up to 96%



# 选型

不同的后缀名有不同的输出选择：

| 型号       | 输出电压            |
| ---------- | ------------------- |
| XM5062_12  | 1.2V                |
| XM5062_18  | 1.8V                |
| XM5062_33  | 3.3V                |
| XM5062_ADJ | 可调（0.6V to Vin） |



更多内容请参考数据手册。
